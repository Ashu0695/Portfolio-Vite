## File Structure
```
/
├── index.html          <- Upload this to public_html root
├── css/
│   └── style.css       <- All styles
├── js/
│   └── script.js       <- All JavaScript
└── images/             <- Add future images here
```

## Hostinger Deployment
1. Login to Hostinger -> Hosting -> Manage -> File Manager
2. Open public_html folder
3. Delete any old files inside public_html
4. Upload this ZIP
5. Extract ZIP directly into public_html
6. Confirm index.html sits directly inside public_html (NOT in a subfolder)
7. Visit your domain - done!

## Netlify Deployment (Easiest)
1. Go to netlify.com -> Sign up free
2. Drag and drop entire folder onto Netlify dashboard
3. Live instantly at free URL
